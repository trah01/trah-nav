# TRAH Nav 隐私政策

**最后更新日期：2026年1月19日**

TRAH Nav（下称“本扩展”）高度重视用户的隐私保护。本隐私政策说明了我们如何处理用户数据。

## 1. 数据收集与使用
TRAH Nav 遵循“本地优先”的设计理念。**我们不会收集、存储或传输您的任何个人身份数据到任何外部服务器。**

*   **本地存储：** 您创建的所有设置、书签和倒计时事件均通过 Chrome 的 `chrome.storage` API 存储在您的本地设备上。
*   **天气数据：** 扩展程序直接与高德地图（AMap）API 通信。您的位置信息（通过 IP）和城市代码仅用于实时获取天气，本扩展开发者无法获取这些信息。
*   **搜索功能：** 搜索请求直接发送至您选择的搜索引擎（如 Google、Bing），受其各自隐私政策约束。

## 2. 权限说明
*   **storage：** 仅用于同步和保存您的个人偏好设置及书签。
*   **newtab：** 仅用于将浏览器新标签页替换为本导航界面。

## 3. 您的权利
由于我们不存储您的数据，因此您拥有对数据的绝对控制权。您可以通过卸载本扩展或清除浏览器数据来删除所有存储的本地配置。
