TRAH Nav 隐私政策

最后更新日期：2026年1月19日

TRAH Nav（下称“本扩展”）高度重视用户的隐私保护。本隐私政策旨在向您说明本扩展如何处理相关数据。

1. 数据收集与使用
TRAH Nav 遵循“本地优先”的设计理念。我们不会收集、存储或传输您的任何个人数据到我们的服务器。

- 用户设置与配置：您创建的所有设置、书签和倒计时事件均通过 Chrome 提供的浏览器存储 API 存储在您的本地设备上。如果您启用了 Chrome 的同步功能，这些数据可能会通过 Google 的加密服务器在您的不同设备间同步。
- 天气数据：为了提供天气预报服务，本扩展会根据您提供的 API 密钥与高德地图（AMap）API 进行通信。本扩展可能会将您的 IP 地址或您指定的城市代码发送给高德地图。我们无法查看、获取或存储这些信息。
- 搜索查询：您在搜索框中输入的内容将直接发送至您选择的搜索引擎（如 Google、Bing）。这些交互受相应搜索引擎服务商隐私政策的约束。

2. 第三方服务
本地数据存储和同步由 Google Chrome 浏览器负责，天气数据由高德地图协议提供。我们建议您阅读这些服务提供商各自的隐私政策。

3. 权限使用说明
- storage（存储）：用于保存您的自定义面板设置、书签和配置信息。
- 新标签页覆盖：本扩展的核心功能逻辑，用于将默认的新标签页替换为个性化导航页面。

4. 政策修订
我们可能会不时更新本隐私政策。任何修订都会通过更新 GitHub 仓库中的隐私政策文件进行发布。

5. 联系我们
如果您对本隐私政策有任何疑问或建议，欢迎通过我们的 GitHub Issues 页面与我们联系：https://github.com/trah01/trah-nav/issues
